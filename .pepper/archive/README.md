# .pepper/archive/

This directory contains archived files that were removed from active use but preserved for historical reference.

## Structure

- `ai-slop/` - AI-generated documentation that was temporary or redundant
- `testing/` - Test reports and artifacts from completed work
- `plans/` - Completed execution plans (moved from .pepper/plans/)

## Retention Policy

- **AI Slop**: Keep for 3 months, then review for deletion
- **Testing**: Keep for 6 months or until related work is stable
- **Plans**: Keep indefinitely as historical record

## When to Archive vs Delete

**Archive if:**
- Contains useful historical context
- May be referenced later
- Documents decisions or learnings
- Unclear if still needed

**Delete if:**
- Completely obsolete
- Contains no unique information
- Duplicates existing documentation
- Build artifacts or generated files

## Adding Files

When archiving, include date in filename: `FILENAME-YYYY-MM-DD.md`
