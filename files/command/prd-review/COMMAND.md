# /prd-review

Review a PRD for completeness and quality.

## Usage

```
/prd-review [prd-path]
```

## What It Does

1. Delegates to **Habanero** for review
2. Checks all sections complete
3. Validates structure against prd-format skill
4. Reports findings and suggestions
